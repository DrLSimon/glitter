Goal

In this tutorial you will complete the file src/glApi.cpp in order to be able to render 2d square made up of two triangles.

Useful files

For this tutorial, you will find the following files useful:

  • src/glApi.hpp
  • src/glApi.cpp
  • shaders/simple2d.v.glsl
  • shaders/simple2d.f.glsl
  • example/PA1Application.hpp
  • example/PA1Application.cpp

Part 1: Completing the API

For each of the following classes (class Buffer), look at the doxygen documentation in the src/glApi.hpp header file and complete the implementation of the class methods in src/glApi.cpp (or directly in the header for templates) so that their implementation checks with the documentation. The list of classes that you need to complete is here:

  • Buffer
  • VAO
  • Shader
  • Program
The remaining classes will be implemented in later assignments.

Part 2: Validation

To verify that your implementation works correctly, you can run: 

$ ./glitter pa1
Rendering after PA1 completion: a yellow rectangle on white background.
